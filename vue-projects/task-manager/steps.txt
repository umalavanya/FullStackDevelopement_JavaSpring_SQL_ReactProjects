step 1.1: Install Node.js, MySQL, VS Code
1.2: Create nuxt project
npm create nuxt-app task-manager
cd task-manager

1.3
install required dependencies
npm install axios jsonwebtoken bcryptjs mysql2

1.4
Create MySQL Database - task-manager-database.MySQL

------------------------------------------
This application has the following features

✅ Frontend: Vue 3 + Nuxt 3 with responsive UI

✅ Backend: Nuxt server API routes

✅ Database: MySQL with users and tasks tables

✅ Authentication: JWT token-based auth

✅ Authorization: Protected routes and task ownership

✅ CRUD Operations: Create, read, update, delete tasks

✅ State Management: Pinia store for auth state
