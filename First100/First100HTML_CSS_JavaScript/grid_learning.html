What is CSS Grid?

CSS Grid Layout is a two-dimensional layout system for the web. It lets you create complex layouts by dividing a page into major regions or defining relationships between HTML elements.

Key Insight: Grid works on two axes (rows AND columns) simultaneously, unlike Flexbox which is one-dimensional (rows OR columns).

Two-Dimensional

Control both rows and columns at the same time. Perfect for complex layouts like dashboards, galleries, and full-page designs.

Precise Placement

Place items exactly where you want them using grid lines, areas, or automatic placement algorithms.

Responsive by Design

Create responsive layouts without media queries using features like auto-fit, minmax(), and repeat().

Grid Item Properties

These properties are applied to grid items (children of the container).

Grid Lines: Grid lines are the dividing lines that make up the grid. They're numbered starting from 1. When you set grid-column: 1 / 3, the item spans from line 1 to line 3.

Real-World Grid Examples

1. Holy Grail Layout (Header, Sidebar, Main, Footer)

.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header"
     "sidebar main"
     "footer footer";
  min-height: 100vh;
}

2. Responsive Gallery (Auto-fit Pattern)

.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
  padding: 20px;
}
auto-fit vs auto-fill: auto-fit expands items to fill available space, while auto-fill keeps empty tracks.

3. Dashboard with Overlapping Elements

.dashboard {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-template-rows: repeat(3, 150px);
  gap: 20px;
}

.card-large {
  grid-column: span 2;
  grid-row: span 2;
}

Grid vs Flexbox

Feature CSS Grid Flexbox
Dimensionality Two-dimensional (rows AND columns) One-dimensional (rows OR columns)
Best For Overall page layout, complex 2D layouts Component layout, aligning items in one direction
Content vs Layout Layout-first approach Content-first approach
Overlap Control Easy item overlapping with grid lines Difficult to overlap items
Browser Support Excellent (all modern browsers) Excellent (all modern browsers)
When to use Grid: Page layouts, dashboards, galleries, complex forms.
When to use Flexbox: Navigation bars, card components, centering items, single rows/columns.
