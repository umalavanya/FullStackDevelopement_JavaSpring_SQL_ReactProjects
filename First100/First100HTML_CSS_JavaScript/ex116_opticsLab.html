This virtual optical lab simulates postgraduate-level physics experiments with realistic optical components and measurement tools. Designed for advanced study of wave optics, interference, diffraction, polarization, and optical system design.

All simulations use physically accurate models with parameters based on real optical components. Data collection includes intensity profiles, interference patterns, and spectral analysis.
