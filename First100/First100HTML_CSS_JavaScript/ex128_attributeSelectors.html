All attribute selector types in one interactive HTML file

1. Presence Selector [attribute]

Selects elements that have the specified attribute

css [data-demo] { border: 2px solid #3498db; }
No data-demo attribute
Has data-demo attribute
Has data-demo="true"

2. Exact Value [attribute="value"]

Attribute value must exactly match

css [data-status="active"] { background: #d4edda; } [data-status="inactive"] { background: #f8d7da; }
Active Item
Inactive Item
Pending Item (no match)

3. Contains [attribute*="value"]

Attribute value contains the substring

css [class*="btn"] { padding: 8px 16px; border-radius: 4px; }

4. Begins With [attribute^="value"]

Attribute value starts with the string

css a[href^="https"] { background: #d1ecf1; } a[href^="http://"] { border-left-color: #dc3545; }
Secure Link (HTTPS)

Insecure Link (HTTP)

Anchor Link (no match)

5. Ends With [attribute$="value"]

Attribute value ends with the string

css a[href$=".pdf"] { color: #e74c3c; padding-right: 20px; } img[src$=".jpg"] { border: 3px solid #f39c12; }
PDF Document

Word Document (no match)

6. Space-Separated List [attribute~="value"]

Attribute value contains word in whitespace-separated list

css [class~="warning"] { border: 2px solid #ffc107; } [class~="success"] { border: 2px solid #28a745; }
Alert with warning class
Message with success class
warning-message (no match - hyphenated)

7. Hyphen-Separated [attribute|="value"]

Attribute value is exactly or begins with value followed by hyphen

css [lang|="en"] { font-family: Arial; } [lang|="es"] { font-family: "Times New Roman", italic; }
English (en)
English US (en-US)
Spanish (es)
Spanish Mexico (es-MX)
english (no match)

8. Case-Insensitive [attribute="value" i]

Attribute value comparison is case-insensitive

css [data-role="admin" i] { background: #4a148c; color: white; }
Admin (lowercase)
ADMIN (uppercase)
Admin (capitalized)
User (no match)

Combined Attribute Selectors

Form Example with Multiple Attribute Selectors

css input[type="text"][required] { border: 2px solid #e74c3c; } input[type="email"][required] { border: 2px solid #3498db; } input[disabled] { opacity: 0.5; }


Link Examples with Multiple Attributes

css a[href^="https"][target="_blank"] { background: #d4edda; border-left-color: #28a745; }
Secure External Link (https + _blank)

Secure Same Window Link (https + _self)

Insecure External Link (http + _blank)
