How to Use:

Tooltips appear on hover and are positioned relative to their trigger element. They include arrow pointers and support multiple styles.

<button class="tooltip-trigger"
    data-tooltip="Your message here"
    data-position="top|bottom|left|right"
    data-style="success|warning|danger">
  Hover me
</button>
