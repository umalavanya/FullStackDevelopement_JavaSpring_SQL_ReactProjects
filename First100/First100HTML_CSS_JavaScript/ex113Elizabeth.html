CSS Techniques Used

CSS Shapes

The face, eyes, hair, and other features are created using CSS border-radius with specific percentages to achieve organic, curved shapes.

Layering & Z-index

Multiple div elements are layered using z-index to create depth - hair behind face, eyes on top, eyelashes on top of eyes, etc.

CSS Animations

Subtle animations bring Elizabeth to life: blinking eyes, gentle smile, floating motion, and sparkling jewelry.

Box Shadows

Multiple box shadows create depth, highlights, and contours - both outer shadows for depth and inner shadows for recessed areas.

CSS Code Snippet 

/* Face - oval shape with gradients */
.face {
    position: absolute;
    width: 180px;
    height: 220px;
    background: #f8dbc1;
    border-radius: 50% 50% 50% 50% / 60% 60% 40% 40%;
    box-shadow: 
        inset 0 -15px 20px rgba(0, 0, 0, 0.1),
        0 5px 15px rgba(0, 0, 0, 0.2);
    z-index: 10;
}

/* Hair - layered elements for volume */
.hair {
    position: absolute;
    width: 220px;
    height: 250px;
    background: #8b4513;
    border-radius: 50% 50% 50% 50% / 60% 60% 40% 40%;
    box-shadow: 
        0 10px 20px rgba(0, 0, 0, 0.3),
        inset 0 -10px 20px rgba(0, 0, 0, 0.2);
    z-index: 5;
}

/* Eyes with blinking animation */
.eye {
    position: absolute;
    width: 30px;
    height: 20px;
    background: #1a1a2e;
    border-radius: 50%;
    z-index: 20;
    animation: blink 5s infinite;
}

@keyframes blink {
    0%, 90%, 100% {
        height: 20px;
    }
    95% {
        height: 5px;
    }
}
