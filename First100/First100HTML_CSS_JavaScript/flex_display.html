CSS Flexbox: Complete Guide

The Modern CSS Layout Module for Flexible and Responsive Designs

What is Flexbox?

Flexbox (Flexible Box Layout) is a CSS layout module that makes it easier to design flexible, responsive layouts without using floats or positioning. It works on a container (flex container) and its children (flex items).

/* Basic Flexbox Setup */
.container {
  display: flex;
}

/* All direct children become flex items */

Flex Container Properties

These properties apply to the flex container (parent element).

flex-direction

Defines the direction of flex items (main axis).

justify-content

Aligns items along the main axis.

align-items

Aligns items along the cross axis.

flex-wrap

Controls whether items wrap to multiple lines.

Flex Items Properties

These properties apply to flex items (children elements).

flex-grow

Defines how much an item can grow relative to others.

0 (no growth)
flex-shrink

Defines how much an item can shrink relative to others.

1 (can shrink)
align-self

Overrides align-items for individual items.

order

Controls the order of items (default is 0).

0 (default order)

Flexbox Cheat Sheet

Container Properties

display: flex - Creates flex container

flex-direction - row | column | row-reverse | column-reverse

justify-content - Main axis alignment

align-items - Cross axis alignment

flex-wrap - Controls wrapping

gap - Space between items

Item Properties

flex-grow - Ability to grow (0 = no, 1 = yes)

flex-shrink - Ability to shrink

flex-basis - Initial size before growing/shrinking

flex - Shorthand (grow shrink basis)

align-self - Override align-items

order - Visual order (default 0)

Common Values

flex: 1 = grow equally

flex: 0 1 auto = default

flex: none = no grow/shrink

flex: 0 0 200px = fixed 200px width

justify-content: center = center horizontally

align-items: center = center vertically

Flexbox vs Other Layout Methods

Feature Flexbox CSS Grid Float/Position
Layout Type One-dimensional (row OR column) Two-dimensional (rows AND columns) One-dimensional, hacky
Best For Components, navigation, small-scale layouts Page layouts, complex grids Simple layouts, wrapping text around images
Browser Support Excellent (all modern browsers) Excellent (all modern browsers) Universal (all browsers)
Complexity Moderate learning curve Steeper learning curve Simple but limited
