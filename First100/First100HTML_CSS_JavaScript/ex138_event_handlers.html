HTML Event Handlers Reference

üåê Window Events

onload
Fires when the page or an element has loaded
<body onload="init()">
onunload
Fires when the page is about to unload
<body onunload="cleanup()">
onresize
Fires when the browser window is resized
window.onresize = function() {
  console.log('Window resized');
}
onscroll
Fires when the element's scrollbar is scrolled
<div onscroll="handleScroll()">
onbeforeunload
Fires before the document is unloaded
window.onbeforeunload = function() {
  return 'Are you sure?';
}

üìù Form Events

onsubmit
Fires when a form is submitted
<form onsubmit="return validate()">
onchange
Fires when an element's value changes
<select onchange="updateSelection()">
oninput
Fires when user inputs data
<input oninput="liveSearch()">
onfocus
Fires when an element gets focus
<input onfocus="showHint()">
onblur
Fires when an element loses focus
<input onblur="validateField()">
onreset
Fires when a form is reset
<form onreset="confirmReset()">
onselect
Fires when text is selected
<input onselect="textSelected()">

üñ±Ô∏è Mouse Events

onclick
Fires on mouse click
<button onclick="handleClick()">
ondblclick
Fires on mouse double-click
<div ondblclick="editItem()">
onmousedown
Fires when mouse button is pressed
<div onmousedown="startDrag()">
onmouseup
Fires when mouse button is released
<div onmouseup="endDrag()">
onmousemove
Fires when mouse moves over element
<canvas onmousemove="draw()">
onmouseover
Fires when mouse enters element
<div onmouseover="showTooltip()">
onmouseout
Fires when mouse leaves element
<div onmouseout="hideTooltip()">
oncontextmenu
Fires on right-click (context menu)
<div oncontextmenu="return false">
onwheel
Fires when mouse wheel is rotated
<div onwheel="zoom(event)">

‚å®Ô∏è Keyboard Events

onkeydown
Fires when key is pressed down
<input onkeydown="checkKey(event)">
onkeyup
Fires when key is released
<input onkeyup="search()">
onkeypress
Fires when key is pressed (character keys)
<input onkeypress="validateChar(event)">

üì¶ Drag & Drop Events

ondragstart
Fires when dragging starts
<div ondragstart="dragStart(event)">
ondrag
Fires while element is being dragged
<div ondrag="drag(event)">
ondragend
Fires when dragging ends
<div ondragend="dragEnd(event)">
ondragover
Fires when dragged over drop target
<div ondragover="allowDrop(event)">
ondragenter
Fires when dragged item enters drop target
<div ondragenter="highlight(event)">
ondragleave
Fires when dragged item leaves drop target
<div ondragleave="unhighlight(event)">
ondrop
Fires when item is dropped
<div ondrop="drop(event)">

üëÜ Touch Events (Mobile)

ontouchstart
Fires when touch starts
<div ontouchstart="touchStart(event)">
ontouchmove
Fires when touch moves
<div ontouchmove="touchMove(event)">
ontouchend
Fires when touch ends
<div ontouchend="touchEnd(event)">

üéµ Media Events

<audio onvolumechange="updateVolumeUI()">
