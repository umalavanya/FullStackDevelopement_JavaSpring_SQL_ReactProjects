Understanding RGB Channels

RGB (Red, Green, Blue) is an additive color model used for digital displays. Each channel has a value from 0 to 255:

• Red + Green = Yellow

• Green + Blue = Cyan

• Blue + Red = Magenta

• All three at maximum (255) = White

• All three at minimum (0) = Black

Understanding HSL Color Model

HSL (Hue, Saturation, Lightness) is a cylindrical color model that represents colors in a more intuitive way:

• Hue: The color type (0°=Red, 120°=Green, 240°=Blue, 360°=Red again)

• Saturation: Intensity of the color (0% = gray, 100% = full color)

• Lightness: Brightness of the color (0% = black, 50% = pure color, 100% = white)

HSL is often easier for humans to understand than RGB because it separates the color (hue) from its intensity (saturation) and brightness (lightness).

Understanding Color Mixing Methods

Additive Mixing (Light): Used for light sources (screens, projectors). Colors add together to create brighter results. Red + Green + Blue = White.

Subtractive Mixing (Paint): Used for pigments, dyes, inks. Colors subtract (absorb) light. Cyan + Magenta + Yellow = Black (in theory).

Average Mixing: Takes the mathematical average of RGB values. Good for understanding color blending.

Multiply Mixing: Multiplies RGB values together (normalized). Darkens colors, similar to layering transparent paints.

Color Wheel Basics

The color wheel shows the relationship between colors:

• Primary Colors: Red, Yellow, Blue

• Secondary Colors: Orange, Green, Violet (mixed from primaries)

• Tertiary Colors: Red-Orange, Yellow-Orange, Yellow-Green, Blue-Green, Blue-Violet, Red-Violet

• Complementary Colors: Opposite on the wheel (e.g., Red & Green)

• Analogous Colors: Next to each other on the wheel (e.g., Red, Red-Orange, Orange)

Understanding White Balance

White balance adjusts colors to make white objects appear white under different lighting conditions.

• Cool (High Kelvin): Bluish tint (6500K+), like daylight or overcast sky

• Neutral (Mid Kelvin): Balanced white (4500K-6500K), like noon sunlight

• Warm (Low Kelvin): Yellow/red tint (2000K-3500K), like candlelight or sunset

• Tint: Adjusts green/magenta balance to compensate for fluorescent lighting
