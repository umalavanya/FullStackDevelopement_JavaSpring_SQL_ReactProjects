3D CSS Properties Explained

transform-style

Determines if children of an element are positioned in 3D space or flattened. Use preserve-3d to create true 3D effects.

perspective

Sets the distance between the user and the z=0 plane. Creates the illusion of depth. Lower values create more dramatic 3D effects.

rotateX/Y/Z

Rotates an element around the X, Y, or Z axis. Combine them to create complex 3D rotations in space.

translateZ

Moves an element along the Z axis (closer or farther from the viewer). Positive values bring elements forward, negative values push them back.
