Inserting HTML Elements with JavaScript

A comprehensive guide to DOM manipulation methods

1. Using createElement() and appendChild()

The most common approach: create an element, modify it, then append it to the DOM.

Code Example

// Create a new div element
const newDiv = document.createElement('div');

// Add content and attributes
newDiv.textContent = "I was created with createElement()";
newDiv.className = "created-div";
newDiv.id = "div-1";

// Append to the result area
document.getElementById('result1').appendChild(newDiv);

2. Using innerHTML

Directly set HTML content of an element. Note: This replaces all existing content.

Code Example

// Using innerHTML to add elements
const container = document.getElementById('result2');

// This REPLACES all existing content
container.innerHTML = `
<div class="created-div">
<h4>Div created with innerHTML</h4>
<p>This entire div was added at once.</p>
<button onclick="alert('Hello from innerHTML!')">Click me</button>
</div>
`;

3. Using insertAdjacentHTML()

Insert HTML at specific positions relative to an element without replacing existing content.

Code Example

// Get the target element
const target = document.querySelector('#result3 h3');

// Insert before the element
target.insertAdjacentHTML('beforebegin',
'<div class="created-div">Inserted BEFORE the title</div>');

// Insert after the element
target.insertAdjacentHTML('afterend',
'<div class="created-div">Inserted AFTER the title</div>');

Positions: 'beforebegin', 'afterbegin', 'beforeend', 'afterend'

4. Comparing append() vs appendChild()

append() can add multiple nodes and text, while appendChild() adds only one node.

Code Example

// Create elements
const div1 = document.createElement('div');
div1.textContent = "First div";
div1.className = "created-div";

const div2 = document.createElement('div');
div2.textContent = "Second div";
div2.className = "created-div";

// append() can add multiple items at once
document.getElementById('result4').append(div1, div2, " Some text node");

// appendChild() can only add one node at a time
// document.getElementById('result4').appendChild(div1);
// document.getElementById('result4').appendChild(div2);
