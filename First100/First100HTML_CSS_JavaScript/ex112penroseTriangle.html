How It Works

3D Transforms

The Penrose Triangle is created using CSS 3D transforms. Each side is a triangular element positioned in 3D space using rotateX(), rotateY(), and translateZ() functions.

Optical Illusion

The "impossible" effect is achieved by positioning additional elements that create the illusion of continuous edges where none actually exist in 3D space.

CSS Clip-Path

Each triangular side is created using CSS clip-path property with a polygon shape. This allows us to create perfect triangles without using images.

CSS Code Implementation 

.penrose-triangle {
    position: relative;
    width: 100%;
    height: 100%;
    transform-style: preserve-3d;
    animation: rotate-triangle 20s infinite linear;
}

.triangle-side {
    position: absolute;
    width: 200px;
    height: 200px;
    background: linear-gradient(45deg, #ff9a9e, #fad0c4, #a18cd1);
    clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
    opacity: 0.9;
}

/* Side 1 - positioned at 0 degrees */
.side-1 {
    transform: rotateX(60deg) rotateY(0deg) translateZ(50px);
}

/* Side 2 - positioned at 120 degrees */
.side-2 {
    transform: rotateX(60deg) rotateY(120deg) translateZ(50px);
}

/* Side 3 - positioned at 240 degrees */
.side-3 {
    transform: rotateX(60deg) rotateY(240deg) translateZ(50px);
}

/* Animation for continuous rotation */
@keyframes rotate-triangle {
    0% {
        transform: rotateX(0deg) rotateY(0deg);
    }
    100% {
        transform: rotateX(360deg) rotateY(360deg);
    }
}
