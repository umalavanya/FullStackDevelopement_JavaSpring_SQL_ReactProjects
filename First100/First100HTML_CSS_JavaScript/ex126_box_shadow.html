üìñ Box-Shadow Property Explained

Parameter Description Required Default Example
offset-x Horizontal shadow position
Positive = right, Negative = left		 Required 0 10px
offset-y Vertical shadow position
Positive = down, Negative = up		 Required 0 5px
blur-radius Blur amount (higher = more blurry) Optional 0 10px
spread-radius Shadow size (positive = expand, negative = shrink) Optional 0 5px
color Shadow color Optional Current text color #000, rgba(0,0,0,0.5)
inset Changes shadow to inner shadow Optional outer inset

üéØ Key Concepts:

  • RGBA colors are best for shadows: rgba(0,0,0,0.1) allows transparency
  • Multiple shadows create depth and realistic effects
  • Negative spread can create "cut-out" effects
  • Performance
  • Use inset shadows for pressed/embossed effects

Text shadow vs box shadow */ box-shadow: 0 2px 5px rgba(0,0,0,0.2); /* for containers */ text-shadow: 2px 2px 4px rgba(0,0,0,0.5); /* for text */
