CSS Selectors Hierarchy

I. BASIC SELECTORS

  • Element tagName (div, p, h1)
    Selects elements by tag name
  • Class .className
    Selects elements with specific class
  • ID #idName
    Selects element with specific ID
  • Universal *
    Selects all elements

II. COMBINATORS

  • Descendant ancestor descendant
    div p - selects all p inside div
  • Child parent > child
    ul > li - direct children only
  • Adjacent Sibling element + nextSibling
    h1 + p - p immediately after h1
  • General Sibling element ~ siblings
    h1 ~ p - all p after h1

III. Attribute Selectors

  • [attribute] Selects elements with the attribute present
  • [attribute = "value"] Exact Match
  • [attribute ~= "value"] Contains word in space-separated list
  • [attribute |= "value"] Value is exactly "value" or begins with "value-"
  • [attribute ^= "value"] Begins with "value"
  • [attribute $= "value"] Ends with "value"
  • [attribute *= "value"] Contains substring "value"
